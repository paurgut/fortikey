Información
¿Qué es Starknet?

Starknet es una red de capa 2 que permite realizar transacciones de forma segura y eficiente en la blockchain de Ethereum.

¿Por qué usar Starknet?
  • Mayor seguridad en el almacenamiento de contraseñas
  • Encriptación descentralizada
  • Sin dependencia de servidores centralizados
¿Cómo funciona?
  1. Conecta tu billetera de Starknet
  2. Genera un token de encriptación único
  3. Usa el token para encriptar tus contraseñas
  4. Guarda el código QR como respaldo
